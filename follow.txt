Deployment Steps

1. Set Up NEAR CLI: Make sure you have the NEAR CLI installed and configured.

Install using the following command: `npm install -g near-cli`

2. Compile the Contract: `cargo build --target wasm32-unknown-unknown --release`

3. Deploy the Contract: near deploy --wasmFile target/wasm32-unknown-unknown/release/<contract_name>.wasm --accountId <your_account.testnet>

4. Initialize the Token: near call <your_account.testnet> new '{"owner_id": "<your_account.testnet>", "total_supply": "1000000000000000000000000"}' --accountId <your_account.testnet>


NB: , "1000000000000000000000000" represents 1 million tokens
